USAGE EXAMPLES
=============

Initialize at app startup:
```kotlin
PerformanceConfig.initialize(context)
```

Check battery before heavy work:
```kotlin
val batteryManager = BatteryOptimizationManager(context)
if (batteryManager.isOptimalForBackgroundWork()) {
    performBackup()
}
```

Profile operations:
```kotlin
suspend fun backup() = profileOperation("Backup") {
    // backup logic
}
```

Lazy loading in lists:
```kotlin
listState.observeScrollToEnd {
    loadMoreItems()
```
