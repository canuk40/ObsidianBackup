                              ARCHITECTURE
================================================================================

Flow:
  External Source → Intent → DeepLinkActivity → DeepLinkHandler
      ↓                                              ↓
  DeepLinkParser ← Parse URI                         ↓
      ↓                                              ↓
  DeepLinkAuthenticator ← Check Auth                 ↓
      ↓                                              ↓
  DeepLinkRouter ← Route Action                      ↓
      ↓                                              ↓
  DeepLinkAnalytics ← Track Event                    ↓
      ↓                                              ↓
  MainActivity ← Final Destination

Key Components:
  • DeepLinkActivity      - Transparent entry point
  • DeepLinkParser        - URI to action conversion
  • DeepLinkAuthenticator - Security verification
  • DeepLinkRouter        - Navigation logic
  • DeepLinkAnalytics     - Usage tracking
  • DeepLinkHandler       - Orchestrates entire flow

================================================================================
                      HTTPS APP LINKS SETUP
================================================================================

1. Generate SHA-256 fingerprint:
   $ keytool -list -v -keystore ~/.android/debug.keystore \
     -alias androiddebugkey -storepass android | grep SHA256

2. Update .well-known/assetlinks.json with fingerprint

3. Host at: https://obsidianbackup.app/.well-known/assetlinks.json
   Requirements:
   - HTTPS only
   - Content-Type: application/json
   - No redirects

4. Verify:
   $ adb shell pm verify-app-links --re-verify com.obsidianbackup
   $ adb shell pm get-app-links com.obsidianbackup
