TECHNICAL ARCHITECTURE
===============================================================================

ObsidianBackup App
       ↓
FilecoinCloudProvider (Kotlin)
       ↓
web3.storage API (HTTP)
       ↓
       ├─→ IPFS Network (Fast Retrieval)
       └─→ Filecoin Network (Long-term Storage)
       ↓
Multiple Public Gateways (Censorship-Resistant)

Data Flow:
Upload:  Device → Encrypt → web3.storage → IPFS → Filecoin Deal → CID
Download: CID → IPFS Gateways → Verify → Decrypt → Device
