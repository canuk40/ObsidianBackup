📋 SETUP IN 3 STEPS
─────────────────────────────────────────────────────────────────────────────────

1. Copy Template
   $ cp local.properties.template local.properties

2. Add Your SafetyNet API Key (edit local.properties)
   safetynet.api.key=AIzaSyXXXXXXXXXXXXXXXXXXXXXXXXX

3. Build
   $ ./gradlew assembleDebug

─────────────────────────────────────────────────────────────────────────────────

🔑 GET SAFETYNET API KEY
─────────────────────────────────────────────────────────────────────────────────

1. Go to https://console.cloud.google.com/
2. Enable "Android Device Verification API"
3. Create API Key
4. Restrict to your app (package name + SHA-1)
5. Copy key to local.properties

🔍 TROUBLESHOOTING
─────────────────────────────────────────────────────────────────────────────────

Build Error: "SAFETYNET_API_KEY not found"
  → Add key to local.properties

Certificate Pinning Failure (SSLPeerUnverifiedException)
  → Pins may be outdated, check SECURITY_SECRETS_MANAGEMENT.md
  → Run: certificatePinningManager.testPinning("googleapis.com")

SafetyNet Attestation Fails
  → Check API key restrictions in Google Console
  → Verify key is enabled for Android Device Verification API

Empty BuildConfig Value
  → Gradle property name: "safetynet.api.key" (with dots)
  → Must be in local.properties at project root
