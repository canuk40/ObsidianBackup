HOW TO USE THE DESIGN SYSTEM
================================================================================

1. SPACING
   Import: import com.obsidianbackup.ui.theme.Spacing
   
   Usage:
     .padding(Spacing.md)            // 16.dp - most common
     .height(Spacing.xs)             // 8.dp - small gaps
     Arrangement.spacedBy(Spacing.lg) // 24.dp - section spacing
   
   Tokens:
     xxxs (2dp)  - Dividers, borders
     xxs  (4dp)  - Minimal gaps
     xs   (8dp)  - Small gaps ← Use this often
     sm   (12dp) - Medium gaps (rare)
     md   (16dp) - Default padding ← PRIMARY
     lg   (24dp) - Section spacing
     xl   (32dp) - Page margins
     xxl  (48dp) - Hero spacing
     xxxl (64dp) - Extra large

2. ELEVATION
   Import: import com.obsidianbackup.ui.theme.Elevation
   
   Usage:
     Card(
         elevation = CardDefaults.cardElevation(
             defaultElevation = Elevation.medium  // 4.dp
         )
     ) { ... }
   
   Levels:
     none    (0dp)  - Flat surfaces
     subtle  (1dp)  - Barely raised
     low     (2dp)  - Inactive cards
     medium  (4dp)  - Standard cards ← DEFAULT
     high    (8dp)  - Active cards, FAB
     highest (12dp) - Modals, dialogs

3. ICON SIZES
   Import: import com.obsidianbackup.ui.theme.IconSize
   
   Usage:
     Icon(
         Icons.Default.Backup,
         modifier = Modifier.size(IconSize.medium)  // 24.dp
     )
   
   Sizes:
     small  (16dp)  - Inline icons
     medium (24dp)  - Standard ← DEFAULT
     large  (48dp)  - List leading icons
     xlarge (64dp)  - Feature icons
     hero   (120dp) - Empty states

4. ANIMATIONS
   Import: import com.obsidianbackup.ui.utils.Animations
   
   Usage:
     // FAB appearance
     AnimatedVisibility(
         visible = condition,
         enter = Animations.fabEnterAnimation,
         exit = Animations.fabExitAnimation
     ) {
         FloatingActionButton(...)
     }
     
     // Loading states
     Crossfade(
         targetState = state.isLoading,
         animationSpec = Animations.crossfadeSpec
     ) { isLoading ->
         if (isLoading) Loader() else Content()
     }
   
   Available Animations:
     - fabEnterAnimation / fabExitAnimation
     - listItemEnterAnimation / listItemExitAnimation
     - slideInFromRight() / slideOutToLeft()
     - slideInFromLeft() / slideOutToRight()
     - crossfadeSpec
     - fadeInAnimation / fadeOutAnimation
     - stepForward() / stepBackward()
     - cardExpandAnimation / cardCollapseAnimation

5. ENHANCED COMPONENTS
   Import: import com.obsidianbackup.ui.components.EnhancedButton
   
   Usage:
     EnhancedButton(onClick = { ... }) {
         Icon(Icons.Default.Backup, contentDescription = null)
         Spacer(modifier = Modifier.width(Spacing.xs))
         Text("Backup")
     }
   
   Features:
     - Haptic feedback (medium on press, light on click)
     - Scale animation (0.95x with spring bounce)
     - Same API as standard Button

6. SEMANTIC COLORS (No More Alpha Hacks!)
   Usage:
     // ❌ BEFORE
     tint = MaterialTheme.colorScheme.primary.copy(alpha = 0.6f)
     color = MaterialTheme.colorScheme.onSurface.copy(alpha = 0.7f)
     
     // ✅ AFTER
     tint = MaterialTheme.colorScheme.primaryContainer
     color = MaterialTheme.colorScheme.onSurfaceVariant
   
   Common Semantic Colors:
     - onSurfaceVariant (for secondary text)
     - outline (for subtle borders)
     - primaryContainer (for tinted icons)
     - errorContainer (for error states)
     - surfaceVariant (for subtle backgrounds)

EXAMPLES - BEFORE & AFTER
================================================================================

Example 1: Simple Card
  BEFORE:
    Card(modifier = Modifier.fillMaxWidth()) {
        Column(modifier = Modifier.padding(16.dp)) {
            Text("Title")
            Spacer(modifier = Modifier.height(8.dp))
            Text("Content")
        }
    }
  
  AFTER:
    Card(
        modifier = Modifier.fillMaxWidth(),
        elevation = CardDefaults.cardElevation(defaultElevation = Elevation.medium)
    ) {
        Column(modifier = Modifier.padding(Spacing.md)) {
            Text("Title")
            Spacer(modifier = Modifier.height(Spacing.xs))
            Text("Content")
        }
    }

Example 2: Empty State
  BEFORE:
    Icon(
        Icons.Default.Backup,
        modifier = Modifier.size(64.dp),
        tint = MaterialTheme.colorScheme.primary.copy(alpha = 0.6f)
    )
  
  AFTER:
    Icon(
        Icons.Default.Backup,
        modifier = Modifier.size(IconSize.hero),
        tint = MaterialTheme.colorScheme.primaryContainer
    )

Example 3: Button with Feedback
  BEFORE:
    Button(onClick = { backup() }) {
        Text("Backup")
    }
  
  AFTER:
    EnhancedButton(onClick = { backup() }) {
        Text("Backup")
    }

Example 4: Loading Animation
  BEFORE:
    if (state.isLoading) {
        CircularProgressIndicator()
    } else {
        Content()
    }
  
  AFTER:
    Crossfade(
        targetState = state.isLoading,
        animationSpec = Animations.crossfadeSpec
    ) { isLoading ->
        if (isLoading) CircularProgressIndicator() else Content()
    }

SUPPORT & QUESTIONS
================================================================================

For Questions:
  1. Check UI_MODERNIZATION_COMPLETED.md for detailed examples
  2. Review UI_MODERNIZATION_PLAN.md for original design intent
  3. See "Known Issues" section if build fails
  4. Refer to this index for quick API reference

Common Questions:
  Q: Which spacing token should I use?
  A: Spacing.md (16.dp) for most padding, Spacing.xs (8.dp) for gaps
  
  Q: How do I fix alpha hacks?
  A: Replace .copy(alpha=0.6f) with semantic colors:
     - primary.copy(alpha) → primaryContainer
     - onSurface.copy(alpha) → onSurfaceVariant
  
  Q: When should I use EnhancedButton?
  A: Primary action buttons (Backup, Restore, Save, etc.)
  
  Q: How do I add animations?
  A: Use Crossfade for loading, AnimatedVisibility for FABs
  
  Q: Build failing with "Conflicting import"?
  A: Remove duplicate Spacing imports (see Known Issues)
