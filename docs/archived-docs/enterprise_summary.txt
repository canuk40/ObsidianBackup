📋 API ENDPOINTS

Authentication:
  POST   /api/v1/auth/login
  POST   /api/v1/auth/saml/initiate
  POST   /api/v1/auth/saml/callback

Devices:
  GET    /api/v1/devices
  POST   /api/v1/devices/register
  GET    /api/v1/devices/{id}
  POST   /api/v1/devices/{id}/wipe
  GET    /api/v1/devices/{id}/compliance

Policies:
  GET    /api/v1/policies
  POST   /api/v1/policies
  PUT    /api/v1/policies/{id}
  DELETE /api/v1/policies/{id}
  POST   /api/v1/policies/{id}/enforce

Audit:
  GET    /api/v1/audit
  GET    /api/v1/audit/{id}
  GET    /api/v1/audit/export

Reports:
  GET    /api/v1/reports/dashboard
  GET    /api/v1/reports/backup-success-rates
  GET    /api/v1/reports/storage-usage
  GET    /api/v1/reports/compliance

  GET    /api/v1/rbac/roles
