INTENT ACTIONS (7)
------------------
✓ ACTION_START_BACKUP
  - Trigger backup with full configuration
  - Parameters: packages, components, compression, incremental, encryption
  - Returns: work_request_id, status, message

✓ ACTION_RESTORE_SNAPSHOT
  - Restore from snapshot
  - Parameters: snapshot_id, optional packages, components
  - Returns: work_request_id, status, message

✓ ACTION_QUERY_STATUS
  - Query operation status
  - Parameters: work_request_id
  - Returns: state, progress, output_data

✓ ACTION_TRIGGER_CLOUD_SYNC
  - Upload to cloud storage
  - Parameters: cloud_provider, optional snapshot_id
  - Returns: work_request_id, status

✓ ACTION_VERIFY_BACKUP
  - Verify backup integrity
  - Parameters: snapshot_id
  - Returns: work_request_id, status

✓ ACTION_CANCEL_OPERATION
  - Cancel running work
  - Parameters: work_request_id
  - Returns: status

✓ ACTION_DELETE_SNAPSHOT
  - Delete backup snapshot
  - Parameters: snapshot_id
  - Returns: work_request_id, status

EVENTS (6)
----------
✓ EVENT_BACKUP_COMPLETE
  - Broadcast on successful backup
  - Includes: snapshot_id, timestamp, apps_backed_up, size, duration, packages

✓ EVENT_BACKUP_FAILED
  - Broadcast on backup failure
  - Includes: apps_failed, message, failed_packages

✓ EVENT_RESTORE_COMPLETE
  - Broadcast on successful restore
  - Includes: snapshot_id, apps_restored, duration, packages

✓ EVENT_RESTORE_FAILED
  - Broadcast on restore failure
  - Includes: snapshot_id, message

✓ EVENT_SYNC_COMPLETE
  - Broadcast on cloud sync success
  - Includes: snapshot_id, bytes_uploaded, duration

✓ EVENT_SYNC_FAILED
  - Broadcast on cloud sync failure
  - Includes: snapshot_id, message

CONTENTPROVIDER QUERIES (4)
----------------------------
✓ content://com.obsidianbackup.tasker/status/{work_id}
  - Query work status by ID
  - Returns: work_id, state, progress, output_data, attempts, tags

  - List recent backups (max 20)
  - Returns: snapshot_id, timestamp, app_count, size, description, status

✓ content://com.obsidianbackup.tasker/latest
  - Get most recent successful backup
  - Returns: same as backups query

✓ content://com.obsidianbackup.tasker/snapshots

  - Returns: snapshot metadata
