# Code Examples

Practical code examples for ObsidianBackup integration and usage.

## Categories

### Basic Operations
- [Basic Backup](basic-backup.md) - Create and manage backups
- Restore Operations - Restore from backups
- Cloud Sync - Synchronize with cloud storage

### Plugin Development
- [Simple Plugin](plugin-example.md) - Basic automation plugin
- Advanced Plugin - Complex plugin with networking
- Custom UI Plugin - Plugin with custom configuration UI

### Integration
- Deep Linking - Launch backups from external apps
- Automation Integration - Integrate with Tasker/Automate
- API Usage - Use ObsidianBackup API programmatically

### Advanced Topics
- Custom Archive Format - Implement custom backup format
- Merkle Tree Usage - Work with Merkle trees
- Encryption - Custom encryption implementation

## Running Examples

Most examples can be run directly in your Android project by:

1. Adding dependency to build.gradle
2. Copying example code
3. Adjusting package names
4. Running on device/emulator
